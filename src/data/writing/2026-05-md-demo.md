## Hva er nytt?

Tidligere var alle postene mine skrevet i JSON-format. Dette gjorde det enklere å samle norsk- og engelskversjonene av posten i en og samme fil. Dette hadde en del begrensninger, blant annet ved bruk av ASCII-elementer i kodeblokker på grunn av newlines.

Alle blogginnlegg og prosjektsider krever en frontmatter-blokk [(les på docs.page)](https://use.docs.page/frontmatter) for titler, hook-en, 'fil-slugs', tags og lesetid.

## Blokktyper

### Kode

```ts
interface Post {
  slug: string;
  body?: ContentBlock[];
}
```

| Blokk     | Kilde        | Komponent       |
| --------- | ------------ | --------------- |
| `heading` | `## tekst`   | `<h2>`–`<h4>`   |
| `list`    | `- item`     | `<ul>` / `<ol>` |
| `callout` | `:::callout` | CardDouble      |
| `details` | `:::details` | Accordion       |

### Callout

:::callout[Tips]
Jeg har også lagt til "callout"-kompenenten fra SRCL-biblioteket, som kan brukes ved å skrive tekst innen en `:::callout[Tittel]`-blokk.
:::

### Trekkspill

:::details[Vis mer]
Dette vises kun når brukeren åpner seksjonen. Jeg legger til en slik ved å skrive tekst i en `:::details[Tittel]`-blokk.
